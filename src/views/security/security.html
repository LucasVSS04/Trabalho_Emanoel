TERMOS E CONDIÇÕES DE USO

1. Introdução

Bem-vindo ao E-Cofre! Este documento estabelece os termos e condições para o uso do nosso sistema de controle financeiro. Ao acessar e utilizar nosso serviço, você concorda com os termos aqui descritos.

2. Definições

Usuário: Qualquer pessoa que utilize a plataforma.
Serviço: O sistema de controle financeiro oferecido.
Dados Pessoais: Informações fornecidas pelo usuário para utilização do serviço.

3. Cadastro e Acesso

O usuário deve fornecer informações verdadeiras e atualizadas para criar uma conta.
A conta é pessoal e intransferível.
O usuário é responsável por manter suas credenciais seguras.

4. Uso do Serviço

O sistema é destinado apenas para gestão financeira pessoal ou empresarial.
O usuário concorda em não utilizar o serviço para atividades ilegais.
O provedor do serviço pode modificar, suspender ou descontinuar funcionalidades a qualquer momento.

5. Privacidade e Proteção de Dados

Os dados fornecidos são protegidos conforme a legislação vigente.
O usuário consente com a coleta e processamento de seus dados para a operação do serviço.
